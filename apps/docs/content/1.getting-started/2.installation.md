# Installation

## Prerequisites

- [Bun](https://bun.sh) v1.3.10 or later
- [Claude Code](https://docs.anthropic.com/en/docs/agents-and-tools/claude-code/overview) CLI installed
- A GitHub or Asana account with API access

## Install from npm

```bash [Terminal]
bunx @pleaseai/agent init --owner YOUR_ORG --title "My Project"
```

The `init` command accepts:

- `--owner <value>` — repository owner (GitHub org or user)
- `--title <value>` — project title
- `--token <value>` — API token (or set `GITHUB_TOKEN` env var)

It will create a GitHub Project and generate a `WORKFLOW.md` file in your repository.

## Manual Setup

If you prefer to set up manually:

### 1. Install the package

```bash [Terminal]
bun add -g @pleaseai/agent
```

### 2. Create WORKFLOW.md

Create a `WORKFLOW.md` file in the root of your target repository:

```markdown [WORKFLOW.md]
---
platforms:
  github:
    kind: github
    api_key: $GITHUB_TOKEN
    owner: YOUR_ORG

projects:
  - platform: github
    project_number: 1

polling:
  interval_ms: 30000

workspace:
  root: ./workspaces

agent:
  max_concurrent_agents: 10
  max_turns: 20
---

You are working on issue {{ issue.title }}.

{{ issue.body }}
```

### 3. Set environment variables

```bash [Terminal]
export GITHUB_TOKEN=ghp_your_token_here
```
